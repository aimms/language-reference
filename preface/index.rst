Preface
-------

.. rubric:: The AIMMS documentation

The AIMMS documentation consists of three manuals

-  *AIMMS-The User's Guide*,

-  *AIMMS-The Language Reference*, and

-  *AIMMS-Optimization Modeling*.

The first two manuals emphasize different aspects in the use of the AIMMS
system, while the third manual is a general introduction to optimization
modeling. All manuals can be used independently.

.. rubric:: Release notes

Which changes and bug fixes are included in particular AIMMS releases
are described in the associated `release notes <https://documentation.aimms.com/release-notes.html>`__.
