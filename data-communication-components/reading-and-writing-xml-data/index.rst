Reading and Writing XML Data
============================

.. |copy|   unicode:: U+000A9 .. COPYRIGHT SIGN

.. rubric:: This chapter

The E\ **x**\ tensible **M**\ arkup **L**\ anguage (XML) is a universal
format for exchanging structured documents and data on the web. For
those unfamiliar with XML, :ref:`sec:xml.intro` provides a short
introduction. It is taken literally from
http://www.w3.org/XML/1999/XML-in-10-points, and is copyrighted
|copy| 1999-2000 by the W3C organization. :ref:`sec:xml.aimms` onwards
explain in detail how AIMMS lets you employ the XML data format from
within your AIMMS applications.

.. rubric:: Further information about XML

If you are unfamiliar with XML, the explanation given here will probably
not be sufficient. The best starting point to obtain further information
about XML, as well as references to specific XML specifications, formats
and tools is the W3C XML site
`http://www.w3.org/XML/ <http://www.w3.org/XML>`__.
