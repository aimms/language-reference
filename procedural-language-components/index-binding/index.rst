Index Binding
=============

This chapter presents the *index binding* rules implemented in AIMMS.
These rules play an essential role during most repetitive set
operations. For standard situations AIMMS behaves as expected. You
should read this chapter if you are interested in a formal discussion of
the rules of the underlying semantics.
